简介

你想认识新朋友吗。Weaving可以帮助人们认识自己想要进入的社交圈的人，拓宽自己的社交圈。

Weaving API还给您或您的公司提供一张人际关系语义网

示例

通过Url搜索

搜索“http://www.google.com/profiles/KaiYanNju“

通过Email搜索

搜索“kaiyannju@gmail.com“

通过关键字搜索

搜索“/search?q=kaiyannju“

其他

项目还在Beta中，如遇不便，请联系kaiyannju@gmail.com。

如果您是开发者，可以在http://code.google.com/p/weaving/参与这个项目
